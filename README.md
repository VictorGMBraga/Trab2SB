Instruções de uso do Montador
-----------

1. Compilar com o seguinte comando: g++ montador.cpp -o montador -std=c++11
2. Executar com o seguinte comando: montador <input>

*** <input> não deve conter a extensão do arquivo (.asm)
*** Precisa da biblioteca elfio 3.1
